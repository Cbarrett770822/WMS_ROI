## üìù **Calculation Formula Pattern**

For most categories:
```
Annual Benefit = (Number of FTEs √ó Cost per FTE) √ó Improvement %
```

For revenue-based:
```
Annual Benefit = (Annual Revenue √ó Operating Margin) √ó Revenue Increase %
```

For cost reduction:
```
Annual Benefit = Annual Cost √ó Reduction %
```
